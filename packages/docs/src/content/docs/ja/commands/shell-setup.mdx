---
title: gh vibe shell-setup
description: 作成した worktree に自動で cd するシェルスニペットを出力する
---

`gh` をラップして、`gh vibe review <PR>` の成功時に自動で作成済みの
worktree に `cd` するシェルスニペットを出力します。

## 設定

シェルの rc ファイルに次の 1 行を追記してください:

```sh
# ~/.bashrc or ~/.zshrc
eval "$(gh vibe shell-setup)"
```

シェルを再読み込みすると:

```sh
gh vibe review 123    # worktree を作成 + 自動 cd
```

## 対応シェル

- `bash`
- `zsh`

その他のシェルでも動作する可能性はありますが、テストはしていません。

## 安全性

shell-setup スニペットは対話シェルを破壊しないよう設計されています:

- ユーザ定義の `gh` 関数 / エイリアスが既に存在する場合、ラッパーは
  インストールを拒否します（標準エラーに警告を出力）。
- ラッパーは `gh-vibe v1` センチネルで囲まれた出力のみを `eval` するため、
  無関係な `gh` 標準出力がシェルコードとして実行されることはありません。
- バイナリ側もシェル用出力を吐くのは、ラッパー経由で起動された
  （`GH_VIBE_SHELL=v1` 環境変数を検出した）**かつ**標準出力が TTY でない
  ときのみです。対話シェルで手動で当該環境変数を設定した場合は、警告付き
  で通常モードにフォールバックします。

## 解除

シェルの rc ファイルから `eval "$(gh vibe shell-setup)"` の行を削除して、
シェルを再読み込みしてください。
